# Ultra Trail de la Ria d'Étel

Site web officiel de l'Ultra Trail de la Ria d'Étel.

## Événement

- **Date** : 23 mai 2027
- **Lieu** : Ria d'Étel, Bretagne
- **Courses** :
  - Ultra 80 km
  - Trail 30 km

## Structure du projet

```
ultra-trail-de-la-ria/
├── index.html          # Page d'accueil
├── css/
│   └── style.css       # Styles principaux
├── js/
│   └── main.js         # JavaScript principal
├── images/             # Images du site
└── pages/              # Pages secondaires
    ├── parcours.html
    ├── inscription.html
    ├── infos-pratiques.html
    └── contact.html
```

## Développement

Site statique HTML/CSS/JS. Ouvrir `index.html` dans un navigateur pour visualiser.

## Hébergement

Compatible GitHub Pages, Netlify, ou tout hébergeur statique.
